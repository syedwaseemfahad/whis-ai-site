Common Questions.

          

              

                  

                      
Is Whis really undetectable?

                      
                  

                  

                      
Yes. Whis uses system-level audio capture and screen OCR that runs entirely outside of the browser. Screenshare tools like Zoom or Teams only capture specific windows or the browser viewport; they cannot detect Whis running in the background as a separate native process.

                  

              

              

                  

                      
Can I get a refund if it doesn't work?

                      
                  

                  

                      
Absolutely. We offer a 7-day money-back guarantee. If the application doesn't work on your machine or you face technical issues we can't resolve, simply email support for a full refund. No questions asked.

                  

              

              

                  

                      
Does it work for Python, Java and C++?

                      
                  

                  

                      
Yes. Whis supports Python, Java, C++, JavaScript, Go, Ruby, and SQL. You can toggle your preferred output language in the settings, and the AI will generate code specifically in that syntax.

                  

              

               

                  

                      
Does it work on Mac and Windows?

                      
                  

                  

                      
Yes, we have native applications for both Windows (10/11) and macOS (Intel & M1/M2/M3 Silicon). Linux support is coming soon.
