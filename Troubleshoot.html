Welcome to Whis-AI

Whis-AI is a powerful general-purpose speech recognition tool. It processes audio and translates it into text with high accuracy. It is optimized for multiple languages and handles technical jargon efficiently.

Key Capability: Whis-AI can perform both Transcription (Audio to Text in same language) and Translation (Audio in any language to English text).

Installation

To use Whis-AI, you need Python installed on your system. It also relies on the command-line tool ffmpeg.

1. Install FFmpeg

This is required to process audio files.

  • Windows: winget install ffmpeg (or via Chocolatey)
  • macOS: brew install ffmpeg
  • Linux: sudo apt update && sudo apt install ffmpeg

2. Install Whis-AI

pip install whis-ai-tool

(Replace with actual package name if different from the standard library).

Quick Start Guide

Command Line Usage

The simplest way to use Whis-AI is through your terminal.

whisper audio.mp3 --model medium

Translate to English

If your audio is in French and you want English subtitles:

whisper audio_french.mp3 --task translate --model medium

Python Script Usage

import whisper

model = whisper.load_model("base")
result = model.transcribe("audio.mp3")
print(result["text"])

Available Models

Whis-AI offers different model sizes. Larger models are more accurate but require more VRAM (GPU memory) and run slower.

Size Parameters VRAM Required Relative Speed
Tiny 39 M ~1 GB ~32x
Base 74 M ~1 GB ~16x
Small 244 M ~2 GB ~6x
Medium 769 M ~5 GB ~2x
Large 1550 M ~10 GB 1x

Advanced Usage

Language Detection

By default, Whis-AI detects the language of the first 30 seconds. To force a specific language, use the language tag:

whisper audio.wav --language German

Output Formats

Whis-AI automatically generates files in the folder where you ran the command:

  • .txt (Plain text)
  • .vtt (Web Video Text Tracks)
  • .srt (SubRip Subtitle file)

Troubleshooting Guide

Encountering issues? Check the solutions below.

Error: "FileNotFoundError: [WinError 2] The system cannot find the file specified"

Cause: FFmpeg is not installed or not added to your system PATH.
Solution: Install FFmpeg (see Installation section) and ensure you can type ffmpeg in your terminal and get a version response.

Error: "CUDA out of memory"

Cause: Your GPU does not have enough VRAM to load the selected model.
Solution: Try using a smaller model. For example, switch from --model large to --model small.

The transcription is repeating the same phrase over and over

Cause: This is a known hallucination issue in silent segments of audio.
Solution: Try increasing the temperature or using Voice Activity Detection (VAD) to filter out silence before processing.

It is running very slowly

Cause: You might be running on the CPU instead of the GPU.
Solution: Ensure you have NVIDIA drivers installed and PyTorch with CUDA support. If you do not have a dedicated GPU, use the tiny or base models for acceptable speed.

Need more help? Check the system logs or contact support at support@whis-ai.com.
